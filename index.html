Como realizar a instalação

Siga o guia rápido abaixo para preparar o aplicativo de estudos no seu dispositivo Windows.

Baixe pelo botão

Use o arquivo executável LuminaSetup.exe servido com segurança por este domínio oficial.

Abra o instalador

Execute o instalador baixado para criar ou atualizar automaticamente a sua instalação local do Lumina.

Entre com sua conta

O beta é restrito. Utilize exclusivamente o endereço de e-mail liberado para os seus testes do Lumina.

Se o Windows bloquear o download

O Windows SmartScreen pode alertar sobre arquivos executáveis novos de forma preventiva.

Confira a origem

O instalador do app deve vir sempre de www.luminaestudos.com.br. Nunca instale links enviados por terceiros.

Certificado SmartScreen

Durante o período de beta, o Windows pode pedir confirmações adicionais até o instalador ganhar reputação na plataforma.

Suporte Técnico

Caso apareça algum erro impeditivo de instalação, envie um print com os detalhes para luminaestudos.app@gmail.com.

Atualizações automáticas

O Lumina Windows beta utiliza o release manifest remoto para identificar publicações de builds novas.

Se uma atualização for obrigatória, o aplicativo bloqueará temporariamente o login, a sincronização de dados, a aba Comunidade e os recursos remotos até que você baixe o instalador mais recente por este site.

Atenção: Se uma atualização for declarada obrigatória, utilize o botão de download acima antes de sincronizar seus dados. Suas informações locais permanecem perfeitamente preservadas no banco de dados local do seu dispositivo.

Acesso controlado de testes

Esta distribuição exclusiva destina-se somente a testadores convidados pelo desenvolvedor.

O Lumina encontra-se em fase ativa de validação técnica, o que significa que ele poderá receber atualizações de versão frequentes, correções de instalação e alterações nas regras de compatibilidade sem aviso prévio.
