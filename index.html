Abra o instalador

O Windows App Installer mostra o nome do app, versão, publicador e permissões.

Entre com sua conta

Suporte

Se aparecer erro de parsing ou instalação, envie print para luminaestudos.app@gmail.com.

Atualizações automáticas

O Lumina Windows beta usa `.appinstaller` para verificar atualizações no início do app. O aplicativo também mantém o version gate Supabase: builds antigas podem ser bloqueadas para login, sync, Comunidade e recursos remotos quando não forem mais seguras.

Se uma atualização for obrigatória, atualize antes de sincronizar dados. Seus dados locais permanecem preservados no dispositivo.

Beta fechado

Esta distribuição é destinada apenas a testers convidados. O Lumina ainda está em validação e pode receber mudanças frequentes de versão, instalação e regras de compatibilidade.
