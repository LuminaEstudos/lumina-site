# Lumina Windows Velopack Releases

Esta pasta recebe o feed gerado pelo `vpk pack`, incluindo arquivos como:

```text
releases.win.json
RELEASES
LuminaEstudos.Lumina-<version>-full.nupkg
```

O conteudo e gerado pelo app repo e copiado automaticamente por:

```powershell
.\tools\lumina_prepare_windows_site_release.ps1 -Force
```
