Como instalar

Baixe pelo botão

Use o arquivo `LuminaSetup.exe` servido por este domínio oficial.

Abra o instalador

O instalador cria ou atualiza a instalação local do Lumina.

Entre com sua conta

O beta é fechado. Use apenas a conta liberada para testes do Lumina.

Se o Windows bloquear

Confira a origem

O instalador deve vir de `www.luminaestudos.com.br`. Não use links enviados por terceiros.

Certificado

Durante o beta, o SmartScreen pode pedir confirmação adicional até o instalador ganhar reputação.

Suporte

Se aparecer erro de bloqueio ou instalação, envie print para luminaestudos.app@gmail.com.

Atualizações automáticas

O Lumina Windows beta usa o release manifest remoto para saber quando existe uma build nova. Se uma atualização for obrigatória, o app bloqueia login, sync, Comunidade e recursos remotos até você baixar o instalador atual.

Se uma atualização for obrigatória, use o botão de download antes de sincronizar dados. Seus dados locais permanecem preservados no dispositivo.

Acesso de teste

Esta distribuição é destinada apenas a testers convidados. O Lumina ainda está em validação e pode receber mudanças frequentes de versão, instalação e regras de compatibilidade.
